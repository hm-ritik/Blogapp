Blog Backend API (FastAPI + Docker)



Hi ,this is my first  backend project  a blog application built using **FastAPI**.
I built this and make a docker image of it by clearning my hands on **Docker** concepts.
Perform Crud Operation Read , Write , Update , Delete. 

Built this using Fastapi write clean code and endpoints which can efficiently perform crud
integrated Database Postgresql (docker-compose.yml)
